What is HTML

HTML stands for Hypertext Markup Language. As a markup language, it serves the purpose of establishing the structure and content of a page. In this sense it is not a programming language like you would think of Python, C, etc. An analogy often used in relation to CSS and JavaScript is that if these three were represented as a building, HTML would be the foundation, columns and pillars, CSS would be the wallpaper and decoration, and JavaScript would be the objects with which you interact inside that building. You could also explain it using a box as an example. HTML would set the structure of that box and the internal divisions of that box. Inside that box you would put the content of the box, still with HTML. With CSS you would stylize the shape of the compartments, the colors, the alignment, etc. With JavaScript you would set how the box opens, how objects move or are interacted with inside the box, etc.

How an HTML file works

The Boilerplate

The boilerplate is the uppermost section of the html document. This section does not appear on the page itself, since it's function is to be an entry point for how a browser opens the page. It sets the document type, important metadata like title, which appears on the browser window, language of the document, important information for SEO purposes, it sets the character set, usually UTF-8, to encompass characters outside of the english lanuage, think latin languages accents or asian languages symbols.

HTML Elements

HTML elements are what determines which type of content is added to the page. This could be a heading, a paragraph, an image, a video, an iframe, etc.

Elements differ in syntax. H1 through H6, P, Section, Div, are elements that need to be closed, while img or video are self closing. The tag to close an element is equal to the opening tag but with a forward slash.
